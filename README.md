# Laravel Sales Management System

Laravelで作られた簡易CRMアプリケーションです.

laravelの個人的な学習のために作成されました.


## 機能

 * 商品のマスタ管理(CRUDとCSVエクスポート)
 * 顧客のマスタ管理(CRUDとCSVインポート)
 * 見積の作成(CRUD)
 * ユーザ情報の管理

## 環境

Homestead 7.18.0で利用しているものをそのまま利用.

 * PHP 7
 * Laravel 5.7
 * MySQL(Heroku上ではPostgres)
